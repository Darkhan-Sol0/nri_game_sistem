## [**Полукровки**](Полукровки):

Игрок выбирает двух родителей из списка чистых рас.

**Особые полукровки:**
- **Тифлинг** = чистая раса + демон (не игровая)
- **Серафим** = чистая раса + ангел (не игровая)
- 
Особые полукровки могут иметь социальный статус (отношение общества) из-за необычного происхождения.

## **Расовые бонусы и недостатки**

**ТЕГИ:**
- *Любой* - Может применить любую особенность, если не противоречит ограничению

### **Полукровка**

*Полукровка (Раса 1 + Раса 2)*
- Конструктор название расы: *Полу(Раса 1)-полу(Раса 2) (Пример: Полуэльф-Полухоббит, Полукошка-Полугном)* / Либо просто полукровка
- Расовый бонус: выбери один расовый бонус из выбранных рас
- *Особенности:* [+1 Особенность *Любой*, +1 Особенность *Теги Расы 1*, +1 Особенность *Теги Расы 2*](Особенности)

*Тифлинг (Дьявол + Раса 2)*
- Расовый бонус: Я дьявол
- *Особенности:* [+1 Особенность *Любой*, +1 Особенность *Дьявол*, +1 Особенность *Теги Расы 2*](Особенности)

*Серафим (Ангел + Раса 2)*
- Расовый бонус: Я ангел
- *Особенности:* [+1 Особенность *Любой*, +1 Особенность *Ангел*, +1 Особенность *Теги Расы 2*](Особенности)
